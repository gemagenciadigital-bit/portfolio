# Eze - Portfolio & Servicios de Marketing Digital Integral

Bienvenido al repositorio oficial de mi portafolio digital. Soy un **Desarrollador Creativo** y experto en **Marketing Digital Integral**.

## 🚀 Sobre este Portafolio
Este sitio web es una experiencia interactiva ("Scrollytelling") construida para demostrar de qué está hecho un verdadero rendimiento visual. Al hacer scroll, podrás ver un recorrido cinemático de alto rendimiento mientras descubres mis servicios y casos de éxito reales.

### Tecnologías:
- **Next.js 14** (App Router)
- **React & TypeScript**
- **Framer Motion** (Sub-scroll animations)
- **Tailwind CSS** (Glassmorphism design)
- **HTML5 Canvas API** (Renderizado fotograma a fotograma dinámico y en máxima resolución por `requestAnimationFrame`).

## 🛠️ Servicios Destacados
No solo construyo sitios web veloces, creo embudos y ecosistemas completos de contenido:
- Desarrollo Web Premium
- Diseño visual para Instagram
- Edición dinámica de REELS
- Campañas escalables en Meta Ads
- Análisis y Estrategia de Mercado
- SEO On-Page/Off-Page

## 💻 Para correrlo de forma local:
Si tienes acceso a este repositorio y deseas previsualizar la página en tu máquina:
```bash
npm install
npm run dev
```
Luego, simplemente abre [http://localhost:3000](http://localhost:3000) en tu navegador.
